# TaskFlow Pro

TaskFlow Pro is a modern task & productivity manager built with Next.js, TypeScript, TailwindCSS, and Zustand.

## Features
- Create, edit, delete tasks
- Priority and completion
- Analytics (Recharts)
- Responsive UI

## Run locally
1. npm install
2. npm run dev

## Tech
Next.js, TypeScript, Tailwind, Zustand, Recharts
